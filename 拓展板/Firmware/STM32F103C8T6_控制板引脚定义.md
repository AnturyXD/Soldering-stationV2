# STM32F103C8T6 拓展板引脚定义 V0.3

本文档记录拓展板 V0.3 固件使用的实际引脚。主控为 STM32F103C8T6，屏幕为 1.3 寸 7Pin ST7789 TFT，负载输出为三路继电器。

## 核心约定

- MCU 逻辑电平为 3.3V。
- ESP-01S、TFT、CH340 调试串口均按 3.3V 逻辑连接。
- 下位机 ATmega 若为 5V 串口输出，进入 STM32 `PB11` 前需要分压或电平转换。
- V0.3 不使用 BEEP、MPU6050、GY-906/MLX90614、OLED。
- 自动模式的人体传感器接口保留，但固件默认不启用自动联动。

## 引脚总表

| 功能 | STM32 引脚 | 方向 | 说明 |
|---|---:|---|---|
| 人体传感器输入 | `PA1` | 输入 | 接口保留，触发电平暂未确认，V0.3 默认不启用 |
| ESP RXD | `PA2 / USART2_TX` | 输出 | STM32 发往 ESP-01S |
| ESP TXD | `PA3 / USART2_RX` | 输入 | ESP-01S 发往 STM32 |
| 联网状态 LED | `PA4` | 输出 | ESP 在线状态指示 |
| TFT SCL/SCK | `PA5 / SPI1_SCK` | 输出 | ST7789 硬件 SPI1 时钟 |
| 状态 LED | `PA6` | 输出 | 预留状态灯，低电平有效时按硬件处理 |
| TFT SDA/MOSI | `PA7 / SPI1_MOSI` | 输出 | ST7789 硬件 SPI1 数据 |
| TFT BLK | `PA8 / TIM1_CH1` | 输出/PWM | 背光控制，V0.3 已支持 PWM 占空比设置 |
| 调试 TX | `PA9 / USART1_TX` | 输出 | CH340 RXD，DMA 非阻塞打印 |
| 调试 RX | `PA10 / USART1_RX` | 输入 | CH340 TXD |
| SWDIO | `PA13` | 调试 | SWD 下载/调试 |
| SWCLK | `PA14` | 调试 | SWD 下载/调试 |
| TFT RES | `PB0` | 输出 | ST7789 复位 |
| TFT DC | `PB1` | 输出 | ST7789 数据/命令选择 |
| BOOT1 | `PB2` | 启动配置 | 建议 10k 下拉到 GND |
| 下位机 TXD | `PB10 / USART3_TX` | 输出 | 预留下发命令到 ATmega |
| 下位机 RXD | `PB11 / USART3_RX` | 输入 | 接收 ATmega 短报文，当前用于在线和温度 |
| 继电器 2：排风扇 | `PB12` | 输出 | 高电平触发，第二路 |
| 继电器 1：照明灯 | `PB13` | 输出 | 高电平触发，第一路 |
| 继电器 3：预留 | `PB14` | 输出 | 高电平触发，第三路 |
| 心跳 LED | `PC13` | 输出 | 低电平点亮 |
| HSE | `PD0/PD1` | 晶振 | 8MHz 外部晶振 |

## 继电器规范

| 继电器 | 引脚 | V0.3 功能 | 默认状态 |
|---|---:|---|---|
| 第一路 | `PB13` | 照明灯 | 关闭 |
| 第二路 | `PB12` | 排风扇 | 关闭 |
| 第三路 | `PB14` | 预留负载 | 关闭 |

固件中三路继电器按高电平有效处理。如硬件改为低电平触发，需要修改 `board_config.h` 中的 `RELAY_ACTIVE_HIGH`。

## 下位机串口

V0.3 使用 `USART3` 接收下位机短报文：

```text
PB10 / USART3_TX -> 下位机 RXD
PB11 / USART3_RX <- 下位机 TXD
baud: 115200
```

当前兼容之前约定的报文：

```text
T:320,S:350,V:24
```

只要 `PB11` 收到完整文本行，就刷新下位机在线时间；超过 `LOWER_ONLINE_TIMEOUT_MS` 未收到数据则判定离线。

## 屏幕

| TFT 引脚 | 信号 | STM32 引脚 |
|---:|---|---:|
| 1 | GND | GND |
| 2 | VCC | 3.3V |
| 3 | SCL | `PA5 / SPI1_SCK` |
| 4 | SDA | `PA7 / SPI1_MOSI` |
| 5 | RES | `PB0` |
| 6 | DC | `PB1` |
| 7 | BLK | `PA8 / TIM1_CH1` |

屏幕无 CS 引脚，SPI1 总线上不要再挂无片选设备。

## 中断优先级

| 中断 | 用途 | 优先级 |
|---|---|---|
| `USART2_IRQn` | ESP-01S 接收 | 抢占 0 / 子 1 |
| `USART3_IRQn` | 下位机接收 | 抢占 0 / 子 2 |
| `DMA1_Channel4_IRQn` | 调试串口 TX DMA | 抢占 1 / 子 0 |
| `DMA1_Channel3_IRQn` | ST7789 SPI1 TX DMA | 抢占 1 / 子 1 |
| `TIM3_IRQn` | 1ms 系统时基 | 抢占 2 / 子 0 |

串口接收优先于显示和调试输出，避免状态报文丢失。
